 # 다은 튜브(DaEunTube) - 개인 프로젝트
 ## 목적
* 아이의 꿈인 유튜버 연습용으로 제작
* 어떠한 웹이던지 게시판은 필수요소

## 개발환경
* Tymeleaf
* Bootstrap
* Spring-Boot 2.5.5
* Spring Data JPA
* QueryDSL
* JDK 11
* MariaDB
* Gradle
