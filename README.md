 # 다은 튜브(DaEunTube) - 개인 프로젝트
 ## 목적
* 딸  연습용으로 제작

* Tymeleaf
* Bootstrap
* Spring-Boot 2.5.5
* Spring Data JPA
* QueryDSL
* JDK 11
* MariaDB
* Gradle
